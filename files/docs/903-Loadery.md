# Wprowadzenie

Loadery służą do pokazania, że coś się dzieje w tle.

W systemie istnieją następujące typy loaderów:
- okrągły
- poziomy

# Duży loader

Główny loader - zasłania cały ekran.

Jest wykorzystywany w modalu.

Uruchamiany przez:
```js
toggleBigLoader();
```
